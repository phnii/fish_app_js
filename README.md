# fish_app_js
 
元はpythonで作成したオリジナルアプリのFishAppを題材に使ってNode.jsとExpressの学習しています。  

#### 実装済み機能  
- ユーザー登録機能
- ログイン・ログアウト機能
- 釣行記投稿機能
- 画像アップロード機能
- 釣行記に紐づく釣果の投稿機能
- 釣行記・釣果の編集機能
- 釣行記・釣果の削除機能
- 釣行記へのコメント機能
- 釣行記一覧表示機能
- 魚名指定による釣行記の検索機能
- グラフ出力機能
- ユーザー間フォロー機能
- ユーザー間DM機能

#### 未実装の機能
- ページネーション機能
- フラッシュメッセージ機能
- テスト
